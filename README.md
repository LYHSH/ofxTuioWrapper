ofxTuioWrapper
==============

Tuio (tuio.org) Wrapper for openFrameworks using ofTouchEvent

ofxTuioClient and ofxTuioServer are tested with oF0071 under Ubuntu 12.04. have a look at tuioExample (client) and mouseToTuioExample (server)

Dependencies
------------
ofxOsc
